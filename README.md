# 智慧檯燈 Smart Desk Lamp

Arduino 智慧檯燈專案，具備多種互動控制功能。

## 功能特色

### 內部控制板 (lamp_internal.ino)
- 🎮 **紅外線遙控** - 使用遙控器調整檯燈角度 (0°/30°/60°/90°)
- 👆 **觸控感應** - 觸控開關控制
- 🚪 **自動開門** - 超音波感測器偵測距離，自動開啟/關閉門
- ⚙️ **伺服馬達** - 控制頂蓋與門的開合

### 外部控制板 (lamp_external.ino)
- 🕐 **時鐘功能** - LCD 顯示日期與時間
- ⏱️ **計時器** - 七段顯示器正/倒數計時
- ⏰ **鬧鐘** - 可設定鬧鐘時間
- 🎵 **音樂播放** - 蜂鳴器播放音樂
- 💡 **LED 控制** - 多種 LED 燈光模式
- 🌍 **時區設定** - 支援 UTC-12 至 UTC+14

## 硬體需求

### 內部控制板
| 元件 | 腳位 |
|------|------|
| 紅外線接收器 | D8 |
| 觸控感應器 | D4 |
| 伺服馬達 (頂蓋) | D3 |
| 伺服馬達 (門) | D5 |
| 超音波 TRIG | D6 |
| 超音波 ECHO | D7 |

### 外部控制板
| 元件 | 腳位 |
|------|------|
| LCD 顯示器 | 8, 9, 4, 5, 6, 7 |
| 蜂鳴器 | D3 |
| 4x4 按鍵矩陣 | A0-A3, D10-D13 |
| 七段顯示器 | D2, D14, D15 |
| 按鈕 | BT1-BT4 |

## 函式庫依賴

```cpp
#include <FlexiTimer2.h>
#include <LiquidCrystal.h>
```

## 按鍵功能說明

| 按鍵 | 功能 |
|------|------|
| 10 | 切換計時器正數/倒數 |
| 11 | 設定計時器數值 |
| 12 | 關閉鬧鐘 |
| 13 | 設定鬧鐘 |
| 14 | 切換時區 |
| 15 | 設定日期與時間 |
